# AuctionCore v2.0 | Real-Time Live Auction System

AuctionCore v2.0 is a full-stack web application designed to eliminate the "Static Web" experience. By utilizing WebSockets, it provides a seamless, zero-latency bidding environment where all users are synchronized in real-time.

## 🚀 Key Features
- **Real-Time Synchronization**: Built with **Socket.io** to ensure bids and activity logs update instantly across all clients without page reloads.
- **Secure Access Protocol**: Features a dual-tab Authentication Overlay (Sign Up/Sign In).
- **Client-Side Validation**: Includes mandatory **@gmail.com** email verification and bid-logic checks to prevent invalid entries.
- **Responsive Dashboard**: A minimalist dark-themed UI built with **CSS Grid** and **Flexbox**, optimized for all screen sizes.
- **Live Activity Feed**: A chronological log of all auction events (user joins, bids placed, system resets).

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Backend**: Node.js, Express.js
- **Communication**: Socket.io (WebSockets)

## 📸 Interface Highlights
- **Image Optimization**: Uses `background-size: contain` to ensure high-performance luxury assets (McLaren P1, iPhone 17 Pro) are displayed perfectly.
- **Security Protocol Page**: Dedicated section explaining the WSS encryption and identity verification.
